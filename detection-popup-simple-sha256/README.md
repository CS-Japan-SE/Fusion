## 概要

シンプルバージョン

ファイル隔離時に、隔離されたファイルのファイルパス（ファイル名含む）とハッシュ値（SHA-256）をポップアップメッセージとして端末に表示するカスタムスクリプトです。

Windows向けです（PopwershellスクリプトをRTRで実行する形になりますので、端末側の環境によってはエラーがでるケースがあります）
